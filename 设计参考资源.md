# 设计参考资源(找素材 + 练审美)

> 用途分两类:① 找素材/抄模式(给 AI 当输入);② 练审美(看 + 拆 + 练)。
> 说明:以下来自 2026-06 的网络检索,我没逐个深度试用,质量/可用性请自己抽样验证。

## 一、能喂 AI 的设计规范(和 refero 一类,最对你的用法)
这类直接产出 AI 编码可读的 `DESIGN.md` / design tokens,放进项目根,Claude Code / Cursor / Codex 等读它生成对应风格 UI。
- **Refero** (refero.design) —— 你已在用,UI 参考 + 生成风格。
- **DESIGN.md Generator** (design.dev/ai/design-md-generator) —— 按开放 DESIGN.md 规范生成,带 WCAG 对比度校验,导出 Tailwind / DTCG / CSS。
- **designtoken.md** (designtoken.md) —— 生成 `designtoken.md` 放项目根,coding agent 当上下文用 token 搭组件。
- **Design AI Systems / DESIGN.md 库** (designmd.app) —— **454 个现成 DESIGN.md**,含 Apple/Stripe/Figma/Slack/Netflix/Airbnb 等大牌风格,直接拿来喂 AI。
- **awesome-design-md** (github.com/VoltAgent/awesome-design-md) —— 大牌设计系统的 DESIGN.md 合集,丢进项目让 agent 生成匹配 UI。
- **Banani** (banani.co) —— 文本生成整套设计系统(token/组件),一键导出 Figma + 代码。

## 六、练审美的方法(光看没用,要拆 + 练)
审美是"看出一个设计为什么好"的能力,可训练:
1. **收集 + 反思**:每周存喜欢的设计,回看时逼自己回答"喜欢哪、为什么"。
2. **拆解**:把你爱的页面拆成排版 / 布局 / 配色 / 间距,看每一处怎么起作用。
3. **抠细节**:圆角、字号、间距这类小处,往往决定"精致感"。
4. **动手仿**:把观察到的手法真做一遍,才真正理解。
5. **观察线下**:招牌、建筑、包装……审美来源不止屏幕。
- 方法类资源:eyefor.design(训练设计眼课程)、readymag 博客《How to train your taste》。
